# AI-Enabled Hybrid LiPo–Li-ion Battery Management System for FPV Quadcopter

## 🚀 Overview

This project presents an AI-enabled hybrid battery management system that integrates LiPo and Li-ion batteries for efficient and safe energy utilization in FPV quadcopters.

The system uses a rule-based expert decision engine to perform real-time, context-aware battery selection based on multiple parameters.

---

## 🧠 AI Features

* Rule-based expert system for decision-making
* Context-aware battery selection (SOC, load, voltage, temperature)
* Mission-aware control (takeoff, cruise, return-to-base, landing)
* Priority-based safety logic for critical conditions
* Hysteresis-based switching for stability

---

## ⚙️ System Components

* MATLAB-based simulation
* Nonlinear battery modeling (SOC, voltage, temperature)
* Real-time telemetry dashboard
* 3D drone visualization
* Weather and disturbance modeling

---

## 📊 Results

Simulation results demonstrate:

* Efficient energy distribution between LiPo and Li-ion batteries
* Stable battery switching behavior
* Safe handling of return-to-base and emergency landing scenarios
